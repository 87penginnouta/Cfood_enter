### 緣起

身處漁產豐富的海島國家，海鮮料理是大家餐桌上的常客。然而看見這些佳餚時，你是否也說不出它的名稱，只記得那是「媽媽的味道」？
本團隊以「食魚文化」為主軸，將二十四種常見海鮮從產地到餐桌的過程，製成網站、遊戲與經營社群，並以國小高年級生為主要受眾，期望讓「好好吃飯」的觀念從小開始，傳承健全的飲食生活與文化，進而珍惜海洋資源。
